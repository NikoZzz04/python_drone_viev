Technologie
<ul>
<li>Python Panel</li>
<li>CesiumJs</li>
</ul>
Uruchamianie
<ol>
<li>Pobierz kod</li>
<li>Otwórz dwie terminale</li>
<li>W pierwszym wpisz python -m http.server 8000</li>
<li>W drugim otwóz środowisko wirtualne</li>
<li>Zaloguj/zarejestruj się w https://cesium.com/platform/cesiumjs/</li>
<li>Tamże pobierz token</li>
<li>Ustaw zmienną CESIUM_ION_TOKEN twoim tokenem</li>
<li>Wpisz panel serve main.py</li>
</ol>
